El Gestor de Seguridad es un sistema que permite la administración, gestión y configuración de los accesos a los usuarios, aplicaciones y diferentes roles a los que se pueden acceder. Por lo tanto Gestor de seguridad permite asegurarse que los recursos del sistema se utilizan de la manera en la que se espera y que quienes puedan acceder a la información que en él se encuentran sean las personas acreditadas para hacerlo.

El Gestor de Seguridad posee las siguientes características:

  • Usuarios:
    		•  Gestión y administracion de usuarios y asignación de roles a cada uno de ellos.
  • Roles:
    		•  Gestión y administración de los diferentes roles, para las dependecnias y el menú de las aplicaciones.
  • Aplicaciones:
    		•  Gestión, administración y creacion de aplicaciones y sus respectivos menús de acceso a los sitemas. Habilitado solo para usuario administrador.
  • Instituciones:
    		•  Gestión, administración y creación de instituciones y dependencias.
